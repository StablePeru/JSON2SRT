# JSON2SRT - Utilidades y Funcionalidades

## Descripción General
JSON2SRT es una aplicación de escritorio que convierte archivos de subtítulos en formato JSON a formato SRT (SubRip Text), añadiendo códigos de color para facilitar la identificación de los personajes principales. Esta herramienta es especialmente útil para editores de vídeo y creadores de contenido que trabajan con subtítulos.

## Funcionalidades Principales

### 1. Conversión de Formatos
- Transforma archivos JSON con información de subtítulos al formato SRT estándar compatible con la mayoría de reproductores de vídeo y software de edición.
- Mantiene la sincronización precisa de los tiempos de entrada y salida de cada subtítulo.

### 2. Identificación de Personajes por Colores
- Analiza automáticamente el diálogo para identificar los personajes principales (hasta 4).
- Asigna códigos de color específicos a cada personaje principal:
  * Personaje principal 1: Amarillo (<AN1>)
  * Personaje principal 2: Azul claro (<CN1>)
  * Personaje principal 3: Magenta (<MN1>)
  * Personaje principal 4: Verde (<VN1>)
  * Personajes secundarios: Blanco (<BN1>)

### 3. Interfaz Gráfica Intuitiva
- Interfaz moderna y fácil de usar desarrollada con PyQt5.
- Soporte para arrastrar y soltar archivos tanto para la entrada como para la salida.
- Selección de archivos mediante botones de exploración.
- Configuración personalizable de FPS (cuadros por segundo) para una conversión precisa de los tiempos.
- Barra de progreso que muestra el avance de la conversión en tiempo real.

### 4. Procesamiento de Texto
- Elimina automáticamente el contenido entre paréntesis del diálogo, que generalmente contiene indicaciones de acción o notas que no son parte del diálogo hablado.

## Requisitos Técnicos
La aplicación está desarrollada en Python y requiere las dependencias especificadas en el archivo requirements.txt.

## Uso Básico
1. Inicie la aplicación ejecutando el archivo principal.
2. Seleccione el archivo JSON de entrada (o arrástrelo y suéltelo en el campo correspondiente).
3. Especifique la ubicación del archivo SRT de salida (se sugiere automáticamente basado en el nombre del archivo de entrada).
4. Ajuste el valor de FPS si es necesario (el valor predeterminado es 25).
5. Haga clic en el botón "Convertir" para iniciar el proceso.
6. Una vez completada la conversión, se mostrará un mensaje de éxito y podrá acceder al archivo SRT generado.

## Estructura del Proyecto
- **src/**: Contiene el código fuente principal
  - **converter.py**: Lógica principal de conversión
  - **main.py**: Punto de entrada de la aplicación
  - **ui/**: Componentes de la interfaz de usuario
  - **utils/**: Utilidades para procesamiento de tiempo, texto y personajes
- **requirements.txt**: Dependencias del proyecto
- **icon.ico**: Icono de la aplicación
